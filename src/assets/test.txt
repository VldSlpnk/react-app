Створити простий застосунок для відображення списку товарів.

Вимоги:
 1. Створити React-застосунок за допомогою Create React App.
 2. Реалізувати наступний функціонал:
 • Отримання даних з API (https://fakestoreapi.com/products).
 • Відображення списку товарів у вигляді карток.
 • Реалізація фільтрації за категоріями.
 • Додавання товарів до кошика.
 • Адаптивна верстка (mobile-first підхід).

Технічні вимоги:
 1. Використовувати:
 • React (хуки).
 • SASS/SCSS для стилів.
 • Git (створити репозиторій на GitHub).
 • Fetch/Axios для роботи з API.
 2. Картка товару повинна містити:
 • Зображення.
 • Назву.
 • Ціну.
 • Категорію.
 • Кнопку “Додати до кошика”.
 3. Верстка:
 • Використання Flexbox/Grid.
 • Адаптація під мобільні пристрої.
 • Дотримання методології БЕМ.
 4. Додатково (буде плюсом):
 • Використання TypeScript.
 • Додавання loader’а під час завантаження даних.
 • Реалізація пагінації.
 • Прості анімації.
 • Можливість генерації статичної сторінки.

Критерії оцінки:
 1. Якість коду:
 • Чистота та читабельність.
 • Структура компонентів.
 • Перевикористання коду.
 2. Верстка:
 • Відповідність макету.
 • Адаптивність.
 • Коректне використання SCSS.
 3. Функціональність:
 • Робота з API.
 • Обробка помилок.
 • Коректна робота фільтрації.

Час на виконання:

3-5 днів

Результат:

Посилання на GitHub-репозиторій з інструкцією щодо запуску проєкту у README.md.
